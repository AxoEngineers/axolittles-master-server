# Axoquarium

The axoquarium can be found at: axoquarium.axolittles.io

The game is hosted in Heroku.

## Setup

```
npm install
```

## Run

the `run.bat` file seems to do the trick, or:

```
node axo.js
```

Once you have the server running, you run the barebones frontend at: 
`http://localhost:3000/`
